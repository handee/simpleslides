JavaScript and HTML5 for interactive graphics:  

HTML5 Slideshow example

  • This is a sample slide show
  • It has not got much content, really
  • There is a sense in which it is a descendent of Rob Flaherty's slideshow; there's a sense in which I've taken his work and ripped out all the fancy nice stuff to make the simplest show ever though.

Aims

  • An absolutely minimal HTML5 slideshow
  • Uses JavaScript (in htmlSlides.js)
  • Uses CSS (styles.css) to style it
  • Has a second CSS to enable easy customisation of look and feel
  • Doesn't use... JQuery. Or anything else like that.
  • One thing it does have that is above and beyond, in this file, is a clock.
  • (That is because I often teach in a lecture theatre with no working clock).
  • To use, you need to have your HTML with a header featuring the navigation stuff (just copy this file)
  • You need to have each slide in a section tag
  • You need the javascript file, and the css file(s)
  • That's it.
